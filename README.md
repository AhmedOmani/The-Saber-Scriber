**The Saber Scribe** is an AI-powered desktop application designed to help English teachers create high-quality educational materials in seconds.

Built for simplicity and speed, it acts as a "Co-pilot" for creating quizzes, vocabulary lists, grammar exercises, and reading passages.

## ‚ú® Key Features

*   **AI Generators:** Instantly create Quizzes, Vocabulary Lists, Grammar Exercises, and Reading Comprehension stories.
*   **Smart Context:** The AI "reads" what you are working on, so you can generate questions based on the story you just wrote.
*   **Rich Text Editor:** A beautiful, distraction-free editor to format your lessons.
*   **AI Refinement:** "Magic" tools to Simplify, Harden, Expand, or Fix Grammar in your text.
*   **Workspace System:** Organize your work into projects (e.g., "Grade 10 - Unit 4") that are automatically saved to your Documents folder.
*   **Export:** Save your work as **PDF** or **Word (.docx)** with one click.
